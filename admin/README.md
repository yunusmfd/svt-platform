# 📝 لوحة الإدارة — محرّر الدروس

أداة محلية لإنشاء دروس جديدة وفق معمارية المنصة (فصل البيانات عن المحتوى).
تُنتج لك **مجلّد الدرس كاملاً** جاهزاً للوضع في `data/lessons/`.

## الوصول

افتح اللوحة عبر **خادم محلّي** (لا عبر `file://`، كي تُحمَّل المستويات):

```bash
python3 -m http.server 8000
# ثم افتح: http://localhost:8000/admin/
```

## ما تُنتجه اللوحة

لكل درس، تولّد اللوحة أربعة مخرجات مطابقة لبنية المنصة:

| المخرج | يوضع في |
|--------|---------|
| `meta.json` | `data/lessons/<id>/meta.json` — البيانات المنظّمة + الاختبار |
| `ar.html` | `data/lessons/<id>/ar.html` — محتوى الدرس بالعربية |
| `fr.html` | `data/lessons/<id>/fr.html` — محتوى الدرس بالفرنسية |
| مدخلة الفهرس | تُضاف إلى مصفوفة `data/lessons/index.json` |

## خطوات الاستعمال

1. **البيانات الأساسية**: املأ المعرّف، ثم اختر المستوى — فتظهر **وحداته** تلقائياً
   من `levels.json` (يضمن ذلك تطابق رقم الوحدة `unitNum`). اختر الفرع والخطاطة والمدّة.
2. **النصوص**: العنوان والوصف وعنوان الفيديو، بالعربية والفرنسية (تبويبان).
3. **المحتوى**: اكتب نصّ الدرس بصيغة HTML مبسّطة في خانتَي «المحتوى (عربي/فرنسي)».
   - استعمل فقرات `<p>…</p>` ووسم `<strong>` للتشديد.
   - لإدراج خطاطة علمية: اختر المفتاح من القائمة واضغط «＋ خطاطة» — يُضاف عنصر
     `<figure data-fig="…">` يحوّله القالب تلقائياً إلى الرسم المناسب.
4. **الاختبار**: أضِف الأسئلة (نصّ + 4 خيارات + الجواب الصحيح + تعليل)، بلغتين.
5. **معاينة فورية** على اليمين تُحدَّث أثناء الكتابة.
6. اضغط **«توليد ملفات الدرس»** — تفتح نافذة فيها المخرجات الأربعة، لكل منها
   زرّا **نسخ** و**تحميل**.

## النشر

1. أنشئ المجلّد `data/lessons/<id>/` وضع فيه `meta.json` و`ar.html` و`fr.html`.
2. أضِف مدخلة الفهرس إلى `data/lessons/index.json` (داخل المصفوفة، بفاصلة).
3. ارفع التغييرات على GitHub — تظهر في المنصة مباشرةً.

## ملاحظات

- **الحفظ التلقائي**: تُحفظ المسودة في `localStorage` أثناء الكتابة؛ إن أغلقت
  الصفحة ورجعت تجدها كما هي. استعمل «توليد ملفات الدرس» للنسخة النهائية.
- **المعرّف فريد** بأحرف لاتينية صغيرة وشرطات فقط (يصبح اسم مجلّد الدرس).
- إن فُتحت اللوحة بلا خادم، تعمل بقائمة مستويات احتياطية دون وحدات — لذا يُفضَّل
  دائماً تشغيلها عبر خادم محلّي.
