# 📝 دليل لوحة الإدارة — محرّر الدروس

## الوصول إلى لوحة الإدارة

على حاسوبك، افتح الملف:
```
nova-svt/admin/index.html
```

أو عبر Live Server:
```
http://127.0.0.1:5500/admin/
```

✅ ستظهر لوحة احترافية شبيهة بـ Blogger.

---

## أجزاء اللوحة

### الجانب الأيسر — محرّر الدرس
- **البيانات الأساسية**: معرّف الدرس، المستوى، الفرع، الخطاطة، المدّة.
- **النصوص**: العربية والفرنسية (عنوان، وصف، فيديو).
- **محتوى الدرس**: محرّر غني يدعم:
  - **عناوين** (H1، H2، H3)
  - **تنسيقات**: عريض، مائل، تسطير
  - **قوائم**: مرقّمة وغير مرقّمة
  - **صور وفيديو**: لصق الروابط مباشرةً
  - **جداول وأكواد برمجية**

### الجانب الأيمن — معاينة فورية
يعرض كيف سيظهر الدرس في المنصة — **يُحدَّث تلقائياً** أثناء الكتابة.

---

## خطوات إنشاء درس جديد

### ١. ملء البيانات الأساسية
| الحقل | المثال |
|-------|--------|
| **معرّف الدرس** | `photosynthese` (بدون مسافات، أحرف صغيرة فقط) |
| **المستوى** | 2bac (الثانية باك) |
| **الفرع** | bio (حياة) |
| **الخطاطة** | cell (خلية) |
| **المدّة** | 45 دقيقة |

### ٢. إدخال النصوص (عربي + فرنسي)
انقر على تبويب **"العربية"** أو **"Français"** وأكمل الحقول:
- **اسم الوحدة**: مثلاً "الوحدة 3: التغذية عند النباتات الخضراء"
- **عنوان الدرس**: مثلاً "التركيب الضوئي"
- **وصف**: جملة قصيرة (تظهر في بطاقة الدرس)
- **عنوان الفيديو**: مثلاً "آلية التركيب الضوئي"
- **طول الفيديو**: بصيغة `MM:SS` مثل `08:30`

### ٣. كتابة محتوى الدرس
في المحرّر الغني (وسط الصفحة):
- اكتب المقدّمة والشرح
- استعمل الأزرار في الشريط العلوي لتنسيق النصّ
- **صور**: انسخ رابط الصورة والصقه (أو استعمل Upload من الشريط)
- **فيديو**: الصق رابط YouTube مباشرةً

### ٤. معاينة
شاهد على اليمين كيف سيظهر في المنصة — **في الوقت الحقيقي**.

### ٥. الحفظ والتصدير

**حفظ مسودة محلياً** (في متصفّحك):
- انقر **"💾 حفظ مسودة"**
- البيانات تُحفظ في `localStorage`
- إذا أغلقت الصفحة ورجعت، ستجد كل شيء محفوظاً

**تصدير JSON** (للنشر في المنصة):
- انقر **"📥 تصدير JSON"**
- نافذة تفتح بـ JSON كامل الدرس
- خيارات:
  - **📋 نسخ**: انسخ الـ JSON والصقه يدويّاً
  - **⬇️ تحميل ملف**: حمّل ملف JSON جاهز

## 🚀 نشر الدرس في المنصة

بعد تصدير JSON:

### الطريقة 1: النسخ اليدوي (الأسهل)
1. اضغط **"📋 نسخ"** في نافذة التصدير.
2. افتح ملف `data/lessons/<المستوى>.json` (مثلاً `data/lessons/2bac.json`).
3. أضف الدرس في القائمة `[ ... ]`:
   ```json
   [
     // ... دروس سابقة ...
     { // ← ضع الـ JSON هنا (الذي نسخته) },
   ]
   ```
4. احفظ الملف.
5. في GitHub/Netlify، ارفع الملف المحدّث.

### الطريقة 2: تحميل الملف (للاحتياطي)
1. اضغط **"⬇️ تحميل ملف"** — يُحمّل `<id>.json`.
2. احفظه احتياطياً في حاسوبك.

## ⚠️ ملاحظات مهمّة

### البيانات المحفوظة محلياً فقط
- لوحة الإدارة تعمل محلياً (بدون server).
- البيانات تُحفظ في `localStorage` متصفّحك فقط.
- **إذا مسحت بيانات المتصفّح، ستفقد المسودات!**
- **حمّل الملف JSON كنسخة احتياطية** دائماً.

### معرّف الدرس يجب أن يكون فريداً
- لا تستعمل نفس الـ ID لدرسين
- استعمل أحرف صغيرة وشرطات فقط: `a-z`, `-`
- أمثلة صحيحة: `photosynthese`, `dna-humain`, `ecosysteme`

### صيغة الفيديو
- اكتب بصيغة **دقائق:ثوان** مثل:
  - `08:30` (8 دقائق و 30 ثانية)
  - `45:00` (45 دقيقة)

### الصور والفيديو
- في المحرّر، يمكنك:
  - لصق **رابط صورة**: `https://example.com/image.jpg`
  - لصق **رابط فيديو YouTube**: `https://www.youtube.com/watch?v=...`

### الأحرف الخاصة
- تجنّب الفواصل والعلامات الخاصة في معرّف الدرس
- استعمل **-** بدل **_** أو **.**

## 💡 نصائح

### تنظيم الدروس
- رتّب الدروس داخل نفس الملف `data/lessons/<level>.json` بترتيب منطقي
- استعمل معرّفات واضحة تعكس محتوى الدرس

### النصوص الطويلة
- استعمل **H2** و **H3** لتقسيم المحتوى
- أضف قوائم بدل فقرات طويلة
- استعمل **blockquote** للنقاط المهمّة

### الصور
- استعمل صور عالية الجودة (لكن محسّنة)
- رابط الصورة يجب أن يبقى متاحاً على الإنترنت
- Imgur أو GitHub rawcontent مناسبان للرفع المؤقّت

## 🔧 استكشاف الأخطاء

| المشكلة | الحلّ |
|--------|------|
| **البيانات اختفت عند تحديث الصفحة** | افتح DevTools → Application → Local Storage → احذف مفتاح `nova-svt-admin-draft` وحاول مجدّداً |
| **JSON خطأ عند النسخ** | تأكّد من ملء جميع الحقول المطلوبة (مع النجمة `*`) |
| **الصورة لا تظهر في المعاينة** | تأكّد أن الرابط يبدأ بـ `http://` أو `https://` |
| **الفيديو لا يعمل** | استعمل رابط YouTube فقط (ليس الرابط القصير) |

## 🎓 مثال عملي

**البيانات الأساسية:**
```
معرّف الدرس: photosynthese
المستوى: tcs
الفرع: bio
الخطاطة: cell
المدّة: 50
```

**النصوص (عربي):**
```
الوحدة: الوحدة 3: التغذية عند النباتات الخضراء
العنوان: التركيب الضوئي
الوصف: كيف تصنع النباتات الخضراء مادتها العضوية من الماء والضوء
عنوان الفيديو: آلية التركيب الضوئي
طول الفيديو: 08:30
```

**المحتوى:**
```
يُعدّ التركيب الضوئي العملية التي تصنع بها النباتات الخضراء...
[كتابة الشرح مع تنسيقات وصور]
```

**التصدير:**
- اضغط "📥 تصدير JSON"
- انسخ الـ JSON
- ألصقه في `data/lessons/tcs.json`
- ارفع على GitHub/Netlify
- الدرس يظهر مباشرةً في المنصة! ✅
