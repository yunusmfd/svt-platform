# Nova SVT — منصة علوم الحياة والأرض

منصة تعليمية رقمية **مجانية** لمادة علوم الحياة والأرض (SVT) وفق المنهاج المغربي،
للسلكين الإعدادي والتأهيلي. ثنائية اللغة (العربية/الفرنسية)، بوضع ليلي، وتعمل
بالكامل كموقع ساكن دون خادم أو قاعدة بيانات أو تسجيل دخول.

> **النسخة الأولى (MVP):** بلا حساب، بلا اشتراك، بلا إعلانات. كل المحتوى متاح للجميع.

---

## ✨ المميّزات

- **دروس منظّمة حسب المستوى والوحدة** وفق المقرّر الرسمي.
- **ثنائية اللغة بنقرة واحدة** (عربية ⇄ فرنسية) مع حفظ الاختيار.
- **وضع ليلي** يتبع تفضيل النظام أو اختيار المستخدم.
- **اختبارات بتصحيح فوري** وشرح بعد كل سؤال ونتيجة نهائية، مع **حفظ أفضل علامة محلياً**.
- **خطاطات علمية** قابلة للتحميل صورةً (PNG)، و**ملخّص نصّي** للدرس.
- **مختبر افتراضي** لملاحظة مراحل الانقسام الخيطي تحت المجهر.
- **تصميم متجاوب** يعمل على الهاتف واللوحة والحاسوب.
- **روابط قابلة للمشاركة** لكل صفحة ومستوى (`#/lessons/2bac`، `#/lesson/...`).
- **يعمل دون إنترنت وقابل للتثبيت** (PWA): بعد أول زيارة تعمل المنصة دون اتصال، ويمكن تثبيتها كتطبيق على الهاتف.

---

## 🧱 التقنيات

بلا أُطُر عمل وبلا أدوات بناء — فقط:

- **HTML5** + **CSS3** (متغيّرات CSS، تصميم متجاوب)
- **JavaScript** خالص (ES Modules)
- **JSON** لتخزين المحتوى
- **SVG** للأيقونات والخطاطات

## 🚀 التشغيل محلياً

المنصة تستعمل **ES Modules** و**fetch** لتحميل ملفات JSON، وهذان لا يعملان عند فتح
`index.html` مباشرةً بنظام الملفات (`file://`). لذا أثناء التطوير يلزم **خادم محلي بسيط**
(أي خادم ساكن يكفي):

**باستعمال Python (متوفّر غالباً):**

```bash
cd nova-svt
python3 -m http.server 8000
```

ثم افتح: `http://localhost:8000`

**أو باستعمال Node:**

```bash
cd nova-svt
npx serve
```

**أو** عبر إضافة **Live Server** في محرّر VS Code (زر «Go Live»).

> ملاحظة: عند النشر على GitHub Pages أو Netlify (عبر HTTPS) لا حاجة لأي خادم —
> يعمل كل شيء مباشرةً.

## 🌐 النشر

المشروع جاهز للنشر **دون أي إعداد أو خطوة بناء**.

### GitHub Pages

1. ارفع المجلّد إلى مستودع على GitHub.
2. من **Settings → Pages**، اختر فرع النشر (مثلاً `main`) والمجلّد الجذر (`/root`).
3. سيُنشر الموقع على `https://<username>.github.io/<repo>/`.

> المسارات في المشروع **نسبية**، لذا يعمل داخل مجلّد المستودع دون تعديل.

### Netlify

- **سحب وإفلات:** اسحب مجلّد المشروع إلى لوحة Netlify.
- **أو** اربط المستودع مع الإعدادات:
  - **Build command:** (اتركه فارغاً)
  - **Publish directory:** `.` (الجذر)
