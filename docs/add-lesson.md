# دليل إضافة الدروس والمستويات

كل محتوى المنصة مخزّن في ملفات **JSON** تحت مجلّد `data/`. يمكنك إضافة درس أو
مستوى جديد بتعديل هذه الملفات فقط، **دون لمس أي كود برمجي**. بعد الحفظ، حدّث
الصفحة في المتصفّح فيظهر المحتوى تلقائياً.

> ⚠️ المهم الوحيد: أن يبقى ملف JSON **صحيح الصياغة** (فواصل، علامات تنصيص
> مزدوجة `"`، ترميز UTF-8). انسخ من القوالب أدناه لتجنّب الأخطاء.

---

## 📁 كيف يُنظَّم المحتوى؟

```
data/
├── levels.json        ← قائمة المستويات الدراسية (13 مستوى، موزّعة على 4 أسلاك)
└── lessons/
    ├── 1ac.json       ← دروس مستوى «الأولى إعدادي»
    ├── 2ac.json
    ├── ...
    └── 2bac-svt.json  ← دروس مستوى «الثانية باك — مسلك علوم الحياة والأرض»
```

- **لكل مستوى ملف دروس واحد** باسم معرّف المستوى (`1ac.json`، `tcs.json`...).
- كل ملف دروس عبارة عن **قائمة (مصفوفة)** من كائنات الدروس `[ {...}, {...} ]`.

---

## ➕ إضافة درس جديد

افتح ملف المستوى المناسب (مثلاً `data/lessons/2bac.json`) وأضِف **كائن درس جديد**
داخل القائمة (لا تنسَ الفاصلة `,` بين الدروس).

### قالب درس كامل (انسخه وعدّله)

```json
{
  "id": "mon-cours",
  "level": "2bac-svt",
  "branch": "bio",
  "fig": "cell",
  "duration": 45,
  "unit":  { "ar": "الوحدة X: ...", "fr": "Unité X : ..." },
  "title": { "ar": "عنوان الدرس", "fr": "Titre du cours" },
  "desc":  { "ar": "وصف قصير يظهر في البطاقة.", "fr": "Brève description." },
  "video": {
    "title":  { "ar": "عنوان الفيديو", "fr": "Titre de la vidéo" },
    "length": "07:20"
  },
  "intro": [
    { "ar": "فقرة تمهيدية. يمكن إبراز كلمة بـ <strong>مهم</strong>.", "fr": "Paragraphe d'introduction." },
    { "ar": "فقرة ثانية اختيارية.", "fr": "Deuxième paragraphe (optionnel)." }
  ],
  "figs": [
    {
      "key": "cell",
      "label": { "ar": "الشكل 1 — ...", "fr": "Fig. 1 — ..." },
      "cap":   { "ar": "تعليق الخطاطة", "fr": "Légende du schéma" }
    }
  ],
  "quiz": [
    {
      "q": { "ar": "نص السؤال؟", "fr": "Texte de la question ?" },
      "options": [
        { "ar": "الخيار الأول",  "fr": "Option 1" },
        { "ar": "الخيار الثاني", "fr": "Option 2" },
        { "ar": "الخيار الثالث", "fr": "Option 3" },
        { "ar": "الخيار الرابع", "fr": "Option 4" }
      ],
      "correct": 1,
      "explain": { "ar": "شرح الإجابة الصحيحة.", "fr": "Explication de la bonne réponse." }
    }
  ]
}
```

### شرح الحقول

| الحقل | النوع | الوصف |
|------|------|------|
| `id` | نص | معرّف **فريد** للدرس، بأحرف لاتينية صغيرة وشُرَط (`mon-cours`). يظهر في الرابط `#/lesson/mon-cours`. |
| `level` | نص | معرّف المستوى، ويجب أن **يطابق** معرّفاً في `levels.json` (وعادةً اسم الملف نفسه). |
| `branch` | نص | `"bio"` لعلوم الحياة أو `"geo"` لعلوم الأرض (يحدّد الأيقونة واللون والتسمية). |
| `fig` | نص | مفتاح الخطاطة المعروضة في بطاقة الدرس. القيم المتاحة في الجدول أدناه. |
| `duration` | رقم | مدة الدرس بالدقائق (مثلاً `45`). |
| `unit` | كائن | اسم الوحدة `{ ar, fr }`. **يجب أن يطابق حرفياً** أحد عناصر `units` الخاصّة بهذا المستوى في `levels.json`، ليظهر الدرس مجمَّعاً تحت وحدته الصحيحة في بوّابة الدروس. |
| `title` | كائن | عنوان الدرس `{ ar, fr }`. |
| `desc` | كائن | وصف قصير للبطاقة `{ ar, fr }`. |
| `video` | كائن | `{ title:{ar,fr}, length:"MM:SS" }` (عرض توضيحي حالياً). |
| `intro` | قائمة | فقرات المقدّمة، كل فقرة `{ ar, fr }`. **يُسمح بوسوم HTML بسيطة** مثل `<strong>`. |
| `figs` | قائمة | الخطاطات العلمية، كل واحدة `{ key, label:{ar,fr}, cap:{ar,fr} }`. |
| `quiz` | قائمة | الأسئلة (انظر أدناه). |

### حقول السؤال (`quiz`)

| الحقل | النوع | الوصف |
|------|------|------|
| `q` | كائن | نص السؤال `{ ar, fr }`. |
| `options` | قائمة | الخيارات، كل خيار `{ ar, fr }` (يُفضَّل 4 خيارات). |
| `correct` | رقم | رقم الخيار الصحيح، **يبدأ من الصفر** (`0` = الأول، `1` = الثاني...). |
| `explain` | كائن | شرح يظهر بعد الإجابة `{ ar, fr }`. |

### مفاتيح الخطاطات المتاحة (`fig` و `figs[].key`)

| المفتاح | الخطاطة |
|--------|--------|
| `cell` | الخلية الحية |
| `dna` | الحمض النووي (ADN) |
| `tecto` | تكتونية الصفائح |
| `gas` | المبادلات الغازية |
| `digest` | الجهاز الهضمي |

> إن استعملت مفتاحاً غير موجود، تُعرض خطاطة الخلية افتراضياً (لا يتعطّل شيء).

## 🆕 إضافة مستوى جديد

1. **أضِف المستوى** إلى `data/levels.json` (قائمة المستويات):

   ```json
   {
     "id": "2ac",
     "stage": "college",
     "code": "2AC",
     "name": { "ar": "الثانية إعدادي", "fr": "2ème collège" }
   }
   ```

   | الحقل | الوصف |
   |------|------|
   | `id` | معرّف فريد قصير (يُستعمل اسماً لملف الدروس). |
   | `stage` | `"college"` (إعدادي)، `"tc"` (جذع مشترك)، `"bac1"` (الأولى باكالوريا)، أو `"bac2"` (الثانية باكالوريا) — يحدّد مجموعته في دليل المستويات. |
   | `code` | الرمز المختصر المعروض على بطاقة المستوى (`2AC`). |
   | `name` | الاسم الكامل `{ ar, fr }`. |

2. **أنشئ ملف دروسه** `data/lessons/<id>.json`. يمكن أن يبدأ بقائمة فارغة:

   ```json
   []
   ```

3. احفظ وحدّث الصفحة — يظهر المستوى تلقائياً كبطاقة في دليل المستويات (`#/lessons`)
   ضمن مجموعة سلكه، ويمكن مشاركته عبر الرابط `#/lessons/<id>`.

> ملاحظة: الصفحة الرئيسية والتذييل يحتويان على **روابط مختصرة ثابتة** لبعض
> المستويات (الأولى إعدادي، جذع مشترك علمي، مسلك علوم الحياة والأرض). المستوى
> الجديد يظهر دائماً في دليل المستويات تلقائياً، أمّا هذه الاختصارات فتُعدَّل
> يدوياً في `index.html` و`assets/js/views/home.js` عند الرغبة.

---

## ✅ نصائح لتفادي الأخطاء

- استعمل علامات التنصيص **المزدوجة** `"` دائماً (JSON لا يقبل `'`).
- ضع فاصلة `,` بين العناصر، و**لا** تضع فاصلة بعد آخر عنصر.
- تأكّد أن `correct` يشير إلى الخيار الصحيح **بدءاً من الصفر**.
- بعد التعديل، يمكنك لصق الملف في أداة فحص JSON على الإنترنت للتأكّد من صحّته.
- وفّر دائماً النصّين العربي `ar` والفرنسي `fr` لكل حقل نصّي.
