# دليل إضافة الدروس والمستويات

تعتمد المنصة معمارية تفصل **البيانات** عن **المحتوى**: بيانات كل درس المنظّمة
في `meta.json`، ومحتواه (النصّ والخطاطات) في ملفَّي `ar.html` و`fr.html`. القالب
الموحّد يقرأ هذه الملفات ويعرض الدرس بنفس التصميم دائماً — فلا يُكرَّر تصميم
الصفحة داخل أي درس.

> 💡 أسهل طريقة لإنشاء درس جديد هي **لوحة الإدارة** (`admin/index.html`): تملأ
> النموذج فتحصل على `meta.json` و`ar.html` و`fr.html` ومدخلة الفهرس جاهزةً.
> ما يلي يشرح البنية يدويّاً.

---

## 📁 كيف يُنظَّم المحتوى؟

```
data/
├── levels.json               ← المستويات الدراسية (13 مستوى على 4 أسلاك) + وحداتها
└── lessons/
    ├── index.json            ← فهرس خفيف: بطاقة كل درس (يُحمَّل عند الإقلاع)
    ├── matiere-energie/       ← مجلّد درس واحد (اسمه = معرّف الدرس)
    │   ├── meta.json          ← البيانات المنظّمة + الاختبار
    │   ├── ar.html            ← محتوى الدرس بالعربية (Fragment HTML)
    │   └── fr.html            ← محتوى الدرس بالفرنسية
    └── .../
```

- **لكل درس مجلّد مستقل** باسم معرّفه، يضمّ ثلاثة ملفات.
- **`index.json`** قائمة (مصفوفة) خفيفة تمنح صفحة المستوى ما تحتاجه لعرض
  البطاقات (عنوان، وحدة، مدّة…) دون تحميل كل الدروس مسبقاً. تفاصيل الدرس
  الكاملة تُجلب كسولاً عند فتحه.

## ➕ إضافة درس جديد (يدويّاً)

### 1) أنشئ مجلّد الدرس `data/lessons/<id>/` وفيه `meta.json`

```json
{
  "id": "mon-cours",
  "slug": "mon-cours",
  "subject": "svt",
  "level": "2bac-svt",
  "unitNum": 2,
  "unit":  { "ar": "الوحدة X: ...", "fr": "Unité X : ..." },
  "order": 0,
  "branch": "bio",
  "thumbnail": "cell",
  "duration": 45,
  "status": "published",
  "title": { "ar": "عنوان الدرس", "fr": "Titre du cours" },
  "desc":  { "ar": "وصف قصير يظهر في البطاقة.", "fr": "Brève description." },
  "video": { "title": { "ar": "عنوان الفيديو", "fr": "Titre de la vidéo" }, "length": "07:20" },
  "quiz": [
    {
      "q": { "ar": "نص السؤال؟", "fr": "Texte de la question ?" },
      "options": [
        { "ar": "الخيار الأول",  "fr": "Option 1" },
        { "ar": "الخيار الثاني", "fr": "Option 2" },
        { "ar": "الخيار الثالث", "fr": "Option 3" },
        { "ar": "الخيار الرابع", "fr": "Option 4" }
      ],
      "correct": 1,
      "explain": { "ar": "شرح الإجابة الصحيحة.", "fr": "Explication de la bonne réponse." }
    }
  ],
  "attachments": [],
  "tags": [],
  "keywords": [],
  "createdAt": "2026-07-01",
  "updatedAt": "2026-07-01"
}
```

| الحقل | النوع | الوصف |
|------|------|------|
| `id` / `slug` | نص | معرّف **فريد** بأحرف لاتينية صغيرة وشُرَط. يظهر في الرابط `#/lesson/<id>` وهو اسم المجلّد. |
| `subject` | نص | ثابت `"svt"`. |
| `level` | نص | معرّف المستوى، يجب أن **يطابق** معرّفاً في `levels.json`. |
| `unitNum` | رقم | **رقم الوحدة** داخل المستوى (من `levels.json`). هو مفتاح تجميع الدرس تحت وحدته. |
| `unit` | كائن | اسم الوحدة `{ ar, fr }` للعرض. |
| `order` | رقم | ترتيب الدرس داخل المستوى (يُستعمل للتنقّل السابق/التالي). |
| `branch` | نص | `"bio"` (حياة) أو `"geo"` (أرض) — يحدّد الأيقونة واللون. |
| `thumbnail` | نص | مفتاح خطاطة بطاقة الدرس (انظر الجدول أدناه). |
| `duration` | رقم | المدّة بالدقائق. |
| `status` | نص | `"published"` أو `"draft"`. |
| `title` / `desc` | كائن | العنوان والوصف `{ ar, fr }`. |
| `video` | كائن | اختياري: `{ title:{ar,fr}, length:"MM:SS" }`. |
| `quiz` | قائمة | الأسئلة التفاعلية (انظر أدناه). |
| `attachments` | قائمة | مرفقات للتحميل: كل عنصر `{ label:{ar,fr}, href, note }`. |

> ملاحظة: **الخطاطات والاختبار مختلفان** — الاختبار تفاعليّ ويبقى في
> `meta.json`، أمّا الخطاطات العلمية فتُكتب داخل محتوى الدرس (انظر الخطوة 2).

### 2) اكتب المحتوى في `ar.html` و`fr.html`

ملفّان يحتويان **Fragment HTML فقط** (بلا `<html>`/`<head>`/`<body>`/سكربت/أنماط):

```html
<p>فقرة تمهيدية. يمكن إبراز كلمة بـ <strong>مهم</strong>.</p>
<p>فقرة ثانية…</p>

<figure class="lesson-fig" data-fig="cell">
  <figcaption>
    <span class="fig-t">الشكل 1 — الخلية</span>
    <span class="fig-c">تعليق مختصر للخطاطة</span>
  </figcaption>
</figure>
```

- استعمل الفقرات `<p>…</p>` ووسم `<strong>` للتشديد.
- **لإدراج خطاطة علمية مشتركة**: أضِف عنصر `<figure class="lesson-fig" data-fig="KEY">`؛
  يحوّله القالب تلقائياً إلى الخطاطة المناسبة مع زرّ تحميل. نصّ `.fig-t` هو
  العنوان و`.fig-c` هو التعليق.
- **لصور خاصّة بالدرس**: أنشئ مجلّد `images/` داخل مجلّد الدرس واستعمل مساراً
  نسبيّاً: `<img src="images/schema.webp" alt="…">`.
- `fr.html` هو نفس المحتوى بالفرنسية. إن غاب، يُعرض `ar.html` كنسخة احتياطية.

### 3) أضِف مدخلة إلى `data/lessons/index.json`

أضِف كائناً إلى المصفوفة (لا تنسَ الفاصلة بين العناصر):

```json
{
  "id": "mon-cours",
  "level": "2bac-svt",
  "unitNum": 2,
  "unit": { "ar": "الوحدة X: ...", "fr": "Unité X : ..." },
  "order": 0,
  "branch": "bio",
  "thumbnail": "cell",
  "duration": 45,
  "hasVideo": true,
  "quizCount": 3,
  "title": { "ar": "عنوان الدرس", "fr": "Titre du cours" },
  "desc": { "ar": "وصف قصير.", "fr": "Brève description." }
}
```

احفظ وحدّث الصفحة — يظهر الدرس بطاقةً تحت وحدته في `#/lessons/<level>`، ويُفتح
عبر `#/lesson/<id>`.

### حقول السؤال (`quiz`)

| الحقل | النوع | الوصف |
|------|------|------|
| `q` | كائن | نص السؤال `{ ar, fr }`. |
| `options` | قائمة | الخيارات، كل خيار `{ ar, fr }` (يُفضَّل 4). |
| `correct` | رقم | رقم الخيار الصحيح **بدءاً من الصفر**. |
| `explain` | كائن | شرح يظهر بعد الإجابة `{ ar, fr }`. |

### مفاتيح الخطاطات المتاحة (`thumbnail` و `data-fig`)

| المفتاح | الخطاطة |
|--------|--------|
| `cell` | الخلية الحية |
| `dna` | الحمض النووي (ADN) |
| `tecto` | تكتونية الصفائح |
| `gas` | المبادلات الغازية |
| `digest` | الجهاز الهضمي |

> إن استعملت مفتاحاً غير موجود، تُعرض خطاطة الخلية افتراضياً (لا يتعطّل شيء).

## 🆕 إضافة مستوى جديد

أضِف المستوى إلى `data/levels.json` مع وحداته:

```json
{
  "id": "2ac",
  "stage": "college",
  "code": "2AC",
  "spec": "green",
  "name": { "ar": "الثانية إعدادي", "fr": "2ème collège" },
  "units": [
    { "num": 3, "spec": "gold", "ar": "الوحدة الثالثة: ...", "fr": "Unité 3 : ..." }
  ]
}
```

| الحقل | الوصف |
|------|------|
| `id` | معرّف فريد قصير. |
| `stage` | `"college"` / `"tc"` / `"bac1"` / `"bac2"` — يحدّد مجموعته في دليل المستويات. |
| `code` | الرمز المختصر على البطاقة (`2AC`). |
| `spec` | لون تخصّص المستوى في بطاقة الدليل (`green/teal/blue/violet/rose/gold`). |
| `name` | الاسم الكامل `{ ar, fr }`. |
| `units` | وحدات المستوى: كل وحدة `{ num, spec, ar, fr }`. `num` هو ما يشير إليه `unitNum` في الدروس. |

المستوى يظهر تلقائياً في `#/lessons`. الدروس تُضاف إليه بإنشاء مجلّداتها +
مدخلاتها في `index.json` (بحقل `level` = معرّف المستوى، و`unitNum` = رقم وحدته).

## ✅ نصائح لتفادي الأخطاء

- استعمل علامات التنصيص **المزدوجة** `"` دائماً في JSON.
- ضع فاصلة `,` بين العناصر، و**لا** تضعها بعد آخر عنصر.
- تأكّد أن `unitNum` في الدرس يطابق `num` وحدةٍ في `levels.json`، وإلا لن يظهر
  الدرس مجمَّعاً تحت وحدته.
- تأكّد أن `correct` يشير إلى الخيار الصحيح **بدءاً من الصفر**.
- ملفّات `ar.html`/`fr.html` تحتوي **Fragment فقط** — بلا وسوم صفحة كاملة أو سكربت.
- وفّر دائماً النصّين العربي والفرنسي، وأبقِ نفس القيم بين `meta.json` والفهرس.
